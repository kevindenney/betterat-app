The identity card stops being a second, different-looking editor — it's a pointer to the same edit-profile screen. No second Sign out (that lives in the dropdown). Inherits the dropdown's orphans: location, blueprints, connected services.
What changed & why

Your question, answered NEW

  • "Your public face" strip sits right under your name — View as others see you opens /sailor/<you> (the real public face), Edit opens edit-profile (photo, bio, position/class/location/club/seasons). Today edit hides behind a tiny pencil and there is no way at all to view your own public face.

Redundancy removed

  • One identity card WAS ×2 — the settings screen's editable header (different style, own photo picker) becomes a compact pointer to the same editor.
  • One Sign out WAS ×2 — dropdown only.
  • One settings entry — "Profile & settings" renamed Account & settings with a subtitle that says what's inside, so the two layers stop sounding like the same thing.

One visual register

  • Both layers use the same iOS inset-grouped grammar (cards, hairlines, chevrons, SF type) — the dropdown stops looking like a different app from the sheet it opens.

Moved down a layer MOVED

  • Location, Subscribed blueprints, Start authoring, Connected services → settings. The dropdown keeps only daily verbs: switch context, Practice, Inbox.

Two layers, confirmed

  • The dropdown is an ephemeral popover — it can't host Units, notifications, subscription and password without becoming a giant scrolling menu. Fast context switcher + durable settings screen is the Apple ID pattern; the bug was the duplication between them, not the layering.

Inbox stays — the badge becomes the shortcut NEW

  • The avatar already wears the unread count, so the signal is always visible without opening anything. Instead of spending a 6th tab or re-adding the header bell, tapping the red badge deep-links straight to Inbox; tapping the avatar opens the menu as usual. Promotion-level speed, zero new chrome.
  • Revisit a real promotion only if Inbox becomes a many-times-a-day surface.
