Library — Groups ↔ Orgs as one continuum

The dial that separates a group from an org is authority — who issues the curriculum. A group is the people you train alongside; an org is the institution that runs the program. Today they read as two unrelated worlds because the yours ↔ stacks toggle (membership) is stacked on top of group ↔ org (scale). The fix: a shared “Your circles” frame, parallel sub-lines, and a parent-org crumb on every group row so the nesting is visible. Pink dots N mark changed copy — see the key below.

What changed (pink pins)

  1. 1
    Parallel sub-lines under each section header. Groups → “The people you train alongside.” Orgs → “The institutions that run the program.” The headers stop being bare nouns and start naming the authority dial. was: no sub-line
  2. 2
    Parent-org crumb on every group row. Owner · RHKYC puts the nesting on screen — the fleet visibly sits inside the club, so “group” and “org” read as room and building, not two worlds.
  3. 3
    Shared “Your circles” super-eyebrow spans Groups + Orgs with a crew → club hairline, framing them as one tight→wide continuum instead of two disconnected zones.
  4. 4
    Both tiers under each segment. “The stacks” now shows Groups to join and Orgs to join, so a region-only fleet isn’t homeless and a joinable club isn’t exiled to a different screen. Membership (yours/stacks) and scale (group/org) become independent axes.
  5. 5
    Interest-native vocab. Same structure, but nursing reads “cohort / school,” sailing reads “fleet / club.” The de-overlap rule: drop “clubs” from the Groups copy (it currently appears in both Groups and Orgs empty states — the root of the fusion). was: “Fleets, clubs, and cohorts you belong to…”
