How to view these

Each .html file is standalone. Open in any browser. The widgets use CSS variables and embedded styles, so they should render correctly on their own. Tabler icons load from CDN.

To regenerate the missing mockups from the continuation session, refer to the Claude conversation transcript or the descriptions in betterat-redesign-spec.md. Each surface's design rationale is documented in the spec.
