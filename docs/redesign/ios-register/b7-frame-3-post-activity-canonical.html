Anatomy · 393 × 852 + spec column

Measurement chips in amber overlay live components. Callout letters (A–G) match Frame 2's positions so the pair reads as a single design-review doc: A is always the state pill, F is always the primary CTA.

Coral retires · green takes a single tick

The only state-shift the user has to read is the swap from a coral pulse to a static green check. Nothing else moves. No confetti, no full-bleed celebration scrim — that would imply "task done, well done" and overshadow the actual next move, which is to reflect on what just happened. The green pill is the entire reward, and it's quiet on purpose.

was · now

Summary card is a bridge, not a destination

The auto-summary appears already-written — a 2-3 sentence reading of what just happened, plus a one-line type breakdown and a Refine summary escape hatch. The card uses the blue-tint family so it visually leans toward the Move to Reflect CTA below. The user can absolutely review and edit, but the affordance whispers: "the work of this frame is done — keep walking."

ios-blue · sparkle eyebrow · gradient tint

One loud action, two quiet ones

Three exits, three weights. Move to Reflect is a solid blue rectangle — gravity. Add another capture is left-aligned blue text — present for the user who isn't done logging. Discard activity is right-aligned gray-1 text — destructive, but visually muted so a wet thumb on a boat won't trip it. Long-press confirmation lives behind it; nothing destroys on a single tap.
