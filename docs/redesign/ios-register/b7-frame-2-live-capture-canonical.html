Do tab, mid-activity: a reverse-chronological capture stream that gets out of the way while the user is in the moment

This is the single frame the user spends the most time inside the Do tab. The phone is held in one hand, screen glanced at between actions. The interface must be calm enough to ignore and loud enough — via the coral LIVE pulse — to confirm "yes, I'm capturing" in a quarter-second glance. Composer is anchored above the tab bar, never blocking the freshest capture.

Anatomy · 393 × 852 + spec column

Measurement chips in amber overlay live components. Each callout is grouped by the same letter (A–G) used in the design-review doc. No spec on this page is invented; every value is in the canonical CSS.

Reverse-chronological is non-negotiable

Mid-activity, the newest capture is at the top — like iMessage, Twitter, or a flight-deck running log. Sailors glance at the most recent thing they said, then keep sailing. A small ↓ Newest first eyebrow makes the orientation explicit. The freshest capture also carries a subtle warm-white wash and a 14% coral ring — confirmation that the last tap landed.

fresh ring · 14% coral · only on the topmost capture

One coral cue, four mixed types

The capture stream mixes voice (italic transcript + 18-bar waveform), typed (roman, 1-line bodies), photo (placeholder with diagonal stripe and monospace label), and time markers (auto-inserted at phase boundaries). Each type is encoded by a 2.5 px left accent — blue, gray, ink — leaving coral free for the state of the surface, not the content of any one capture.

voice · typed · photo

Two bottom anchors, not one

The composer (tap-to-add) and the stop-CTA are separate rows. The composer is the continuous action — "add another" — and stays one tap away at all times. The stop-CTA is the terminal action — "this activity is done" — and uses reverse polarity (white fill, coral border) so the user reads it as serious but not alarming. Stacking the two protects against accidental commits while keeping voice capture truly one-handed.
