5Migration order

Cheap visual standardizations first; structural extraction once the visual contract is settled.

  1. Atlas: move avatar right, add +, swap title-text for an interest pill. Single PR, no other tabs touched.
  2. Flip TabScreenToolbar's default to interestSwitcherLeft=true so Discover/Watch/Library agree with Practice without per-call overrides.
  3. Unify inbox icon — every tab uses NotificationBell size={20}; remove the custom action-button variant from Discover/Watch.
  4. Extract AppChromeRow — CanvasTopBar collapses into it; TabScreenToolbar's nav row delegates to it; Atlas TopChrome delegates to it.
  5. Build ScopeRow with breadcrumb + zoom rail + vitals chips. Drop into Practice L2/L3/L4 and Library zone pages.
  6. L1 chrome restore — add the chrome row to the step view and migrate the status pill into Vitals.
  7. Retire CanvasTopBar once steps 4–5 land.
