One FAB, two paths, three taps to a plan

The Practice tab carries exactly one floating affordance: a 56pt iOS-blue plus, anchored 16pt from the right edge and 16pt above the tab bar. It is the only entry point for adding steps. Tap once → sheet. Tap Build with AI Coach → conversation. Send one message → draft. Three taps minimum, no form to stare at.

iOS blue · 56pt diameter · two-layer shadow tinted blue

The sheet has exactly two options

An iOS-native action sheet using grouped frosted cards: Build with AI Coach (primary, blue icon tile) and From a Blueprint (secondary, gray icon tile). Each option carries a sub-label so the user understands the path before tapping. Cancel sits below, full-width, separated by an 8pt gap — standard iOS pattern.

Conversation produces a draft, not a form

The AI Coach screen looks like Messages but reads as a creation tool: gray assistant bubbles with a sparkle-gradient avatar, blue user bubbles right-aligned. After two exchanges the assistant emits a draft Plan card inline — same field structure as the canonical Plan tab, compressed to ~75% scale. The user accepts (blue primary) or keeps refining (text button). No "submit" — just commitment or continued conversation.
